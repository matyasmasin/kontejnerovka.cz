## Definice semaforu

- Zelena: web roste nebo je technicky v poradku.
- Oranzova: neni chyba, ale chybi data nebo je potreba neco zlepsit.
- Cervena: problem muze brzdit indexaci, poptavky, duveru nebo mereni.
